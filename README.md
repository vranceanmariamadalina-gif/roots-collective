# Roots Collective

Marketing site for **Roots Collective** — immersive learning & development experiences where cooking becomes the room for reflection, collaboration, and trust.

A single-page site with the styles and scripts split into separate files.

## Structure

```
index.html    Page markup
style.css     All styles
script.js     Nav, accordion, carousels, scroll reveal
```

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```

## Deploy

Hosted with **GitHub Pages** from the `main` branch root.
